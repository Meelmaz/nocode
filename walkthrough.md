# Proyecto NeuroIA — Inteligencia Artificial y Redes Neuronales

¡El proyecto está terminado! He desarrollado una plataforma web **Full Stack** completa utilizando **Django** y **Python**, con un diseño futurista de alto nivel.

## 🚀 Cómo ejecutar el proyecto

1. El servidor ya debería estar corriendo en segundo plano.
2. Abre tu navegador y ve a: `http://127.0.0.1:8000/`
3. Para entrar al panel privado, primero debes **Registrarte** o usar una cuenta existente.

---

## 🛠️ Tecnologías Utilizadas

- **Backend**: Python 3.12 + Django 6.0 (Framework robusto y seguro).
- **Frontend**: HTML5, CSS3 (Custom Glassmorphism), JavaScript (Vanilla).
- **Estilos**: Bootstrap 5 (Base) + CSS personalizado para el look futurista.
- **Base de Datos**: SQLite (Integrada por defecto en Django).
- **Diseño**: Inspiración en *Cyberpunk* y *High-Tech* con colores neón verde (#00ff88) y rosa (#ff4fd8).

---

## 📂 Estructura del Proyecto

- `proyecto_ia/`: Configuración principal de Django (settings, urls).
- `web/`: Aplicación principal que contiene la lógica (views, models, forms).
- `templates/`: Archivos HTML divididos en componentes y páginas.
- `static/`:
    - `css/styles.css`: Todo el diseño futurista, animaciones y glassmorphism.
    - `js/main.js`: Lógica de scroll, efectos 3D en las tarjetas y partículas.
    - `images/`: Imágenes 3D generadas específicamente para este proyecto.

---

## 🧠 Funcionalidades Implementadas

1. **Sistema de Autenticación**: 
   - Registro de usuarios con validaciones.
   - Login seguro utilizando el sistema de sesiones de Django.
   - Logout y protección de rutas (solo usuarios logueados ven el Dashboard).
2. **Diseño Premium**:
   - **Glassmorphism**: Efectos de cristal esmerilado en navbar y tarjetas.
   - **Cards 3D**: Las tarjetas de "Tipos de Redes" se inclinan siguiendo el movimiento del ratón.
   - **Animaciones**: Scroll suave, apariciones graduales (Fade Up) y orbes de luz animados en el fondo.
   - **Responsive**: Totalmente adaptado para móviles y tablets.
3. **Contenido Educativo**:
   - Definiciones históricas de IA (McCarthy, Minsky, Russell).
   - Explicación visual de capas y funcionamiento de Redes Neuronales.
   - Catálogo detallado de arquitecturas (CNN, RNN, LSTM, GAN, etc.) con sus respectivas imágenes 3D.

---

## 🎤 Presentación del Desarrollador

Este proyecto demuestra el poder de **Django** para manejar la lógica de negocio y seguridad (Backend) de forma eficiente, mientras que el **Frontend** se encarga de proporcionar una experiencia de usuario (UX) impactante y moderna.

- **Django** se encarga de recibir las peticiones, procesar los formularios de usuario, guardar los datos en la base de datos y renderizar los templates con la información correcta.
- El **CSS** utiliza variables modernas y filtros de desenfoque para lograr esa estética tecnológica "premium".
- El **JavaScript** añade esa capa extra de interactividad que hace que la web se sienta "viva".
