Redes Neuronales Artificiales

Comprendiendo la arquitectura inspirada en el cerebro humano

¿Qué son?

Las Redes Neuronales Artificiales (RNA) son modelos computacionales inspirados en la estructura y funcionamiento del cerebro humano. Están compuestas por unidades de procesamiento llamadas neuronas artificiales, organizadas en capas y conectadas entre sí mediante pesos sinápticos.

¿Cómo funcionan?

Cada neurona recibe entradas, las multiplica por sus pesos, aplica una función de activación y produce una salida. El proceso de aprendizaje consiste en ajustar los pesos de las conexiones para minimizar el error entre la salida obtenida y la salida deseada, utilizando algoritmos como backpropagation (retropropagación del error).

Capas Neuronales

Capa de Entrada

Recibe los datos iniciales del exterior. Cada neurona representa una característica del dato de entrada.

Capas Ocultas

Procesan la información mediante transformaciones matemáticas. Pueden ser una o varias capas (deep learning).

Capa de Salida

Produce el resultado final de la red. El formato depende del tipo de problema (clasificación, regresión, etc.).

Tipos de Redes Neuronales

Conoce las principales arquitecturas y sus aplicaciones

Básico

Perceptrón Simple

La unidad más básica de una red neuronal. Es un modelo de una sola neurona que puede clasificar datos en dos categorías mediante una función de activación lineal. Fue propuesto por Frank Rosenblatt en 1958.

Clasificación binaria Lineal
Intermedio

Perceptrón Multicapa (MLP)

Extensión del perceptrón con múltiples capas ocultas. Puede resolver problemas no lineales gracias a las funciones de activación no lineales. Utiliza backpropagation para el entrenamiento.

No lineal Backpropagation
Avanzado

CNN — Convolucional

Diseñada para procesar datos con estructura de cuadrícula como imágenes. Utiliza capas convolucionales que aplican filtros para detectar patrones como bordes, texturas y formas complejas. Revolucionó la visión por computadora.

Visión por computadora Filtros
Avanzado

RNN — Recurrente

Diseñada para datos secuenciales. Las neuronas tienen conexiones que forman ciclos dirigidos, permitiendo que la red tenga "memoria" de entradas anteriores. Ideal para series temporales y procesamiento de lenguaje.

Secuencias Memoria temporal
Avanzado

LSTM — Long Short-Term Memory

Variante especial de RNN que resuelve el problema del desvanecimiento del gradiente. Utiliza compuertas (forget, input, output) para controlar el flujo de información y mantener dependencias a largo plazo.

Memoria largo plazo Compuertas
Generativo

GAN — Generativa Adversaria

Arquitectura compuesta por dos redes: un Generador que crea datos sintéticos y un Discriminador que intenta distinguir datos reales de falsos. Compiten entre sí mejorando continuamente. Propuesta por Ian Goodfellow en 2014.

Generación de imágenes Adversarial
